Table with Custom Sort Types

This sortable table uses two custom sort types that are defined in two external Javascript files called numberksorttype.js and uscurrencysorttype.js. These define the sort types NumberK and UsCurrency. The NumberK sort type uses the scientific modifiers K, M, G and T (the computer science ones, not the general science ones) when comparing the value by number. The UsCurrency takes a string representing a US currency formatted with optional commas ($4,000,123.99)
